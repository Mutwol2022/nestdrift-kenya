# Contributing to NestDrift Kenya

Thank you for your interest in improving NestDrift Kenya! Here's how to get involved.

## Ways to Contribute

- Report bugs (wrong prices, broken buttons, layout issues)
- Suggest new features via GitHub Issues
- Submit improvements via Pull Requests
- Add more Kenyan counties, areas, or estates to the filters

## How to Submit a Bug Report

1. Go to the **Issues** tab on GitHub
2. Click **New Issue**
3. Describe what happened and what you expected
4. Include which browser you used (Chrome, Firefox, Safari, etc.)

## Code Contributions

1. Click **Fork** on the top right of this repository
2. Make your changes to `index.html`
3. Test it locally by opening in your browser
4. Submit a Pull Request with a clear description of your changes

## Style Guide

- All prices must be in **KES**
- All location names must be real Kenyan counties, areas, or estates
- Keep the single-file structure (`index.html`) — no splitting into multiple files unless agreed
